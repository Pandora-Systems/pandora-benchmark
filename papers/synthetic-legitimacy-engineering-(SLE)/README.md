# Synthetic Legitimacy Engineering (SLE)

**Status:** under development  
**Repository area:** `papers/synthetic-legitimacy-engineering-(SLE)/`

Synthetic Legitimacy Engineering is a developing Pandora research line focused on how AI systems respond to perceived legitimacy inside structured interactions.

The central question is simple:

> How do artificial systems decide that a request, role, frame, or task appears legitimate enough to follow?

SLE treats legitimacy as an interaction signal rather than a static truth. It studies how authority, context, procedure, necessity, and framing can shape model behavior under evaluation conditions.

This directory is reserved for future public-facing material related to the theory, vocabulary, and analytical boundaries of SLE.

Operational methods, sensitive prompt structures, and reproducible elicitation procedures are not included here.

## Current release state

## Boundary

SLE is intended as an analytical and safety-research framework, not a prompt library, jailbreak guide, or operational attack manual. Because legitimacy modeling can have dual-use implications, this directory currently provides only a controlled research placeholder while the framework remains under development.
